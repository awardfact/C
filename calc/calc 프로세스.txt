후위표기을 담는 배열과  입력받을 값을 담는 배열 입력받은 값을 숫자로 바꾼값이 들어갈 배열 선언
해당 배열들의 길이를 담는 변수 출력 

사용자에게 계산을 입력받음 
입력받은 값들 중에 숫자를 정수형으로 바꿔주는 atoi함수 실행 
-atoi함수   (atoi2함수 사용 )
입력받은 문자 반복문을 돌린다 
해당 문자가 식에 사용되는 문자가 아니면 생략 
문자가 숫자라면 해당 숫자가 어느정도 크기인줄 모르기 떄문에 임시 변수에 숫자를 넣고 전 문자가 숫자라는것을 체크
문자가 숫자가 아닌 경우 전의 문자가 숫자면 배열에 숫자를 추가해준다 그리고 해당 문자를 배열에 추가해준다 
반복이 끝난 경우에 마지막 문자가 숫자였다면 해당 숫자를 배열에 추가해준다 

정수로 변환한 식을 후위식으로 변환시켜주는 mtob함수 실행 
-mtob함수    (pushEst popEst 함수    , rank함수 실행 )
정수로 변환학 식을 반복 돌린다 
연산자인 경우
연산자 스택이 비어있으면 해당 연산자를 push한다
연산자 스택이 비어있지 않으면 해당 연산자의 우선순위와 스택 top의 우선순위를 비교한다
해당 연산자의 우선순위가 더 높으면push 아니면 pop을 하는데 pop을 한 경우 스택에 더 남아있으면 반복한다 
pop을 하는 경우에는 후위표기에 연산자를 추가한다
피연산자인경우 
후위표기에 피연산자 추가 
반복이 종료되고나서 연산자 스택에 값이 남아있다면 pop을 해서 후위표기에 추가한다 

후위표기로 변환하고 후위표기를 계산해서 출력하는 calc함수 실행 
-calc함수  (pushi , popi함수 실행)
후위표기식을 반복 돌린다 
피연산자가 나오면 피연산자 스택에 숫자를 push한다
연산자가나오면 피연산자 스택에서 2개의 값을 pop한(앞에 값이 두번쨰 pop) 후 두 피연산자를 가지고 해당 연산을 실행하고 
나온 값을 다시 피연산자 스택에 push한다 
반복이 끝나고 숫자 스택에서 pop을 한다 이 값이 연산 결과이다
